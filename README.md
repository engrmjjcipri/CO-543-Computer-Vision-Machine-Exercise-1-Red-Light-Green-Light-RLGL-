# CO543-Computer Vision-Machine Exercise-1: Red Light-Green-Light (RLGL)
Real-time Red Light Green Light game using webcam-based motion detection. Detects player movement from video frames movement is allowed during Green Light, but any motion during Red Light results in a “dead” state.

# 🧠 Main technologies used

CSS → styling (layout, colors, progress bars, overlays)

JavaScript (Vanilla JS) → main program logic
